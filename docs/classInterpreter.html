Detailed Description

Class Interpreter - Interpreter of plain text files with description for generating sculptures.

Helper class for reading and interpreting files that describe a sculpture in a certain expected model. It captures the contents of a plain text file and generates a sequence of geometric shapes, children of the GeometricFigure class. This sequence that can be used to build a three-dimensional digital model instantiated by Sculptor class.
